# Pico W Chip Temperature Measurement

This project demonstrates how to measure the chip temperature using a **Raspberry Pi Pico W**.

---

## Prerequisites

Before running the project, make sure you have the following installed:

- [Git](https://git-scm.com/)
- [CLion](https://www.jetbrains.com/clion/) or [VSCode](https://code.visualstudio.com/)
- ARM GCC toolchain
- [Ninja build system](https://ninja-build.org/)
- A debugger compatible with Pico W or without
